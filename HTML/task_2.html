Favicon Tutorial

How to design a favicon using favicon.cc

As follows :To add a favicon to your website, either save your favicon image to the root directory of your webserver, or create a folder in the root directory called images, and save your favicon image in this folder. A common name for a favicon image is "favicon.ico".

      Next, add a  element to your "index.html" file, after the "title" element.

            Now, save the "index.html" file and reload it in your browser. Your browser tab should now display your favicon image to the left of the page title.
            Use the HTML  element to insert a favicon.

  • OPen www.favicon.cc
  • Create favicon
  • Save/Download the favicon into HTML folder
  • Or Copy the URL
  • Add image or link in head element

  • Save the code
  • Run the file through Live server
